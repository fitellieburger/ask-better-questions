# Ask Better Questions
### A classroom-ready tool for the media literacy mandate

---

11 states passed new media literacy laws since January 2024. Most lack implementation tools.

**The gap is real.** Legislators created the requirement. Curriculum directors inherited it. Teachers are expected to build habits — habits that media professionals spend years developing — with no purpose-built tools to support them. Existing resources were designed before algorithmic news feeds existed.

**Ask Better Questions (ABQ) closes that gap.** It's a free tool that works on any article, on any device, in any class period — with no setup, no accounts, and no student data collected.

---

## How it works

- **Students** paste any article URL or text into ABQ — in the browser or via Chrome extension
- **ABQ generates** three sets of prompts: questions that probe claims, questions that surface what's missing, and search cues for what to look up next
- **Each prompt includes a "why"** — a brief explanation tied to that specific article, ready for class discussion

No summaries. No verdicts. Students still have to read, think, and decide.
